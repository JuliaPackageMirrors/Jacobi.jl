# Jacobi

A library that implements Jacobi polynomials $P_n {\alpha,\beta}(x)$ and Gauss 
quadrature related operations.

## Jacobi polynomials

Jacobi polynomials $P_n {a,b}(x)$ are implemented in function 
`jacobi(x, n, a, b)`
